# Doorknob

Using a Raspberry Pi to control a door, using your voice

***WORK IN PROGRESS***

## Requirements

* A Raspberry Pi with an internet connection
* Python3
* NodeJS
* gpiozero

## Documentation

### Set up

1. Clone repository
2. Create a `.env` file with a port to communicate over
3. `$ node app.js`

### Updating

1. Open repo in the terminal
2. `$ git pull`
